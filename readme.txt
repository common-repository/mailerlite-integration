== Description ==

Easy to use integration of MailerLite(mailerlite.com) E-mail marketing tool.

= Features =
* Easy and quick configuration
* Shortcode to create customizable subscription forms
* Tested with WordPress 3.4.2

= Supported parameters for shortcode =
* define the group for subscription form
* customize form labels, values
* customize error, success messages on forms

= System requirements =
* To use this plugin you need PHP SoapClient class

== Installation ==

Unzip the plugin and copy the mailerlite-integration folder to your wp-content/plugins folder or install from wordpress.org repository and activate the plugin.

== Frequently Asked Questions ==

= Where can I find the API Key? =

Login to your MailerLite account, on the bottom of the page you will find a link "Developer API" click on it and on the next page you will see your API Key.
